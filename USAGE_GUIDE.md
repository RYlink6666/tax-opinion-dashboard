## 📊 各页面详解

### **P1 - 总体概览** 📈
**核心目标**: 30秒内掌握舆论数据概况

**展示内容**：
- 📊 **关键指标卡片** (5张)
  - 总舆论数、正面比例、风险舆论数、主流话题、核心参与方
- 📈 **4个维度的分布**
  - 情感分布（正/中/负）
  - 风险分布（安全/低/中/高/严重）
  - 话题分布（Top 10）
  - 参与方分布（消费者/企业/政府等）

**适用用户**：
- ✅ 决策者（需要快速汇报）
- ✅ 新用户（初次了解数据）
- ✅ 高管（月度审查）

**使用建议**：
```
1. 打开P1，查看5张指标卡片（30秒）
2. 滚动查看4个分布图（1分钟）
3. 若发现异常指标，点击相应页面深入分析
```

### **P2 - 意见搜索** 🔍
**核心目标**: 用多条件过滤找到你需要的舆论

**交互功能**：
- 🔎 **关键词搜索** (支持全文搜索)
  - 在原文中搜索任意关键词（如"税收"、"关税"、"政策"）
- 📋 **多维度筛选**
  - 情感（正面/中立/负面）
  - 风险等级（安全/低/中/高/严重）
  - 话题（跨境电商税收 / 监管态度 / 企业成本 等）
  - 参与方（消费者 / 政府机构 / 电商企业 等）
- 📊 **结果展示**
  - 表格形式展示搜索结果（源文本、情感、风险、话题等）
  - 支持排序和分页

**典型应用场景**：
```
场景1：找所有"负面 + 高风险"的舆论
  → 情感选"负面"，风险等级选"高/严重" → 搜索

场景2：查找与"企业成本"相关的所有舆论
  → 关键词输入"成本" → 搜索

场景3：看消费者对"跨境电商税收"话题的意见
  → 参与方选"消费者"，话题选"跨境电商税收" → 搜索
```

**使用建议**：
```
1. 明确你的搜索目标（是找特定话题？找高风险舆论？）
2. 使用筛选器缩小范围（可同时选多个维度）
3. 点击表格中的行查看完整原文
```

### **P3 - 风险分析** 🚨
**核心目标**: 发现最危险的舆论，了解风险分布

**分析维度**：
- 🔴 **风险等级分布** (饼图)
  - 每个风险等级的舆论数量占比
- 🔥 **高风险舆论特征**
  - 高风险/严重舆论中，哪种情感最多？
  - 高风险舆论与话题的关系（热力图）
  - 高风险舆论与参与方的关系（热力图）
- 📝 **高风险舆论示例**
  - 展开器显示具体的高风险舆论原文及特征

**关键数据**：
```
- "严重"风险舆论数：X条
- 最高风险参与方：XX（Y条严重舆论）
- 最高风险话题：XX（Z条高风险舆论）
```

**使用建议**：
```
如果你是政策制定者：
  1. 查看"严重"和"高"风险的舆论数量
  2. 看热力图，找出"高风险 + XX话题"的组合
  3. 点开示例，阅读几条最危险的舆论
  4. 制定针对性的政策应对措施

如果你是企业：
  1. 关注"你企业 + 高风险"的舆论数量
  2. 看这些高风险舆论在说什么
  3. 制定舆情应对方案
```

---

### **P4 - 模式分析** 📊
**核心目标**: 理解不同交易模式下的舆论差异

**分析维度**：
- 📈 **各模式的舆论数量** (柱状图)
- 😊 **各模式的情感倾向** (堆积图)
  - 每个模式中，正/中/负面舆论的比例
- ⚠️ **各模式的风险特征** (热力图)
  - 不同模式中，各风险等级的舆论分布
- 💬 **各模式的典型案例**
  - 展开器显示各模式下的代表性舆论

**关键问题**：
```
Q: 哪个模式舆论最积极？
→ 看柱状图 + 堆积图，找出正面占比最高的模式

Q: 哪个模式风险最大？
→ 看热力图，找出"严重+高风险"最多的模式

Q: 我应该鼓励哪个模式？
→ 综合看舆论数量、情感倾向、风险等级，选择正面且低风险的模式
```

**使用建议**：
```
1. 查看柱状图，确认各模式的舆论规模
2. 看堆积图，比较各模式的情感倾向（哪个最积极？）
3. 看热力图，找出最有争议/风险的模式组合
4. 点开示例，了解具体舆论内容
```

---

### **P5 - 参与方分析** 👥
**核心目标**: 理解消费者、企业、政府等不同群体的观点

**分析维度**：
- 📊 **参与方舆论数量** (柱状图)
- 😊 **参与方情感倾向** (堆积图)
  - 消费者vs企业vs政府，谁最正面？谁最负面？
- ⚠️ **参与方的风险特征** (热力图)
  - 不同参与方中，哪些人群制造的高风险舆论最多？
- 💬 **参与方的代表观点**
  - 展开器显示各参与方的典型舆论

**关键洞察**：
```
消费者观点：
  - 正面舆论多？ → 消费者对政策满意
  - 高风险舆论多？ → 消费者对政策有强烈反感

政府/企业观点：
  - 态度如何？ → 支持还是反对新政策？
  - 有没有高风险言论？ → 需要紧急回应吗？
```

**使用建议**：
```
如果你需要了解某个群体的态度：
  1. 在P5找到这个参与方（如"消费者"）
  2. 看堆积图，查看其情感分布
  3. 看热力图，了解其风险特征
  4. 点开示例，读几条代表性舆论

如果你想比较多个群体：
  1. 看柱状图，对比舆论规模
  2. 看堆积图，对比情感倾向
  3. 看热力图，对比风险特征
  4. 找出最积极/最消极/最危险的群体
```

### **P6 - 政策建议** 💡
**核心目标**: 将舆论分析转化为决策建议

**包含内容**：
- 📌 **关键发现**（基于舆论数据）
  ```
  1. 整体舆论态度：X% 正面，Y% 负面
  2. 核心关切：XX、XX、XX（出现频率最高的话题）
  3. 风险预警：高风险舆论集中在XX领域
  ```

- 💼 **利益相关方观点总结**
  ```
  消费者：主要关心 XX、XX、XX
  企业：主要关心 XX、XX、XX
  政府机构：观点 XX
  ```

- ✅ **政策建议**（3-5条）
  ```
  建议1：针对消费者最关心的"XX"，采取...
  建议2：对于高风险的"XX话题"，应该...
  建议3：参考国际经验，可以考虑...
  ```

- 📊 **实施指标**
  ```
  - 实施前基线：当前负面舆论 X%
  - 目标状态：6个月内负面舆论降至 Y%
  - 跟踪指标：每周监测高风险舆论数量
  ```

**使用场景**：
```
场景1：向领导汇报
  → 把P6的内容直接用于演讲稿

场景2：制定政策方案
  → 参考"政策建议"和"利益相关方观点"

场景3：评估政策影响
  → 实施政策后，用"实施指标"跟踪效果
```

**使用建议**：
```
1. 先读"关键发现"，了解整体舆论态势
2. 读"利益相关方观点"，理解各群体的真实想法
3. 评估"政策建议"的可行性
4. 建立"实施指标"进行后续跟踪
```

### **P7 - 话题热度敏感度分析** 🔥
**核心目标**: 找出最热的话题、最敏感的话题、最有争议的话题

**分析维度**：
- 📊 **话题热度** (柱状图)
  - 各话题的舆论数量（讨论最多的话题）
- 🔴 **话题敏感度** (热力图)
  - 各话题中，高/严重风险舆论的占比
  - 某话题 + 某风险等级的舆论数量
- 😊 **话题情感分布** (堆积图)
  - 各话题中，正/中/负面舆论的比例
  - 哪些话题最容易引发负面舆论？

**关键应用**：
```
Q: 消费者在讨论什么？
→ 看热度排名，找Top 3话题

Q: 哪个话题最容易出现风险舆论？
→ 看敏感度热力图，找"高风险占比"最高的话题

Q: 我应该重点关注哪个话题？
→ 综合考虑热度 + 敏感度 + 情感，选择"讨论热烈但有风险"的话题

Q: 某话题是否安全讨论？
→ 看情感分布，如果大多是正面，可以积极引导讨论
→ 如果多是负面，需要谨慎沟通
```

**使用建议**：
```
1. 看柱状图，找出舆论最热的5个话题
2. 看敏感度热力图，找出风险最高的话题
3. 看情感分布，了解每个话题的舆论倾向
4. 针对"热度高 + 敏感度高"的话题，制定特殊沟通策略

如果你是舆论管理：
  - 定期监测这7个话题
  - 当某话题敏感度上升时，提前预警
  - 准备回应方案，以防万一
```

### **P9 - 互动分析工具** 🔬
**核心目标**: 深入分析单条舆论，理解其背景和含义

**功能**：
- 🔍 **舆论选择**
  - 按话题、参与方、情感等条件筛选舆论
  - 或输入关键词搜索
- 📊 **单条详情**
  - 完整原文、来源、发布时间
  - 自动识别的话题、情感、风险等级、参与方、交易模式
- 🏷️ **标签管理**（高级功能）
  - 自定义添加/编辑标签
  - 对标签进行再分析

**典型应用**：
```
场景1：快速判断一条舆论
  1. 输入关键词或选择条件
  2. 看自动分析结果（话题、情感、风险）
  3. 阅读原文，判断是否准确
  4. 如果有异议，手动调整标签

场景2：深入研究某个舆论背景
  1. 选中感兴趣的舆论
  2. 查看其所有字段（参与方、话题、情感等）
  3. 与其他类似舆论进行对比
  4. 理解其代表的观点倾向

场景3：质量控制
  1. 随机抽样几条舆论
  2. 验证自动分析是否准确
  3. 如发现系统性错误，反馈给数据团队
```

**使用建议**：
```
这是最细粒度的分析页面，适合：
  - 深度研究某些舆论
  - 质量验证（检查自动标签是否准确）
  - 撰写案例分析或学术论文时查找具体证据
```

## ⚙️ 常见问题 FAQ

### Q1: 各页面的数据是同步的吗？
**A**: 是的。所有页面都使用同一份数据源，经过相同的清洗和标签流程。如果P1显示"1000条舆论"，那所有其他页面也是基于同一1000条。

### Q2: 数据多久更新一次？
**A**: 目前是静态数据（2,297条固定舆论）。未来计划每月自动爬取新数据并更新。

### Q3: 可以导出数据或报告吗？
**A**: P2（意见搜索）的表格可以直接复制数据。P6（政策建议）的内容可以截图或复制用于报告。我们计划在后续版本加入"一键生成PDF报告"功能。

### Q4: 如果我发现数据或分析有误？
**A**: 
- 数据标签错误（如某条舆论的"情感"或"话题"标错了）？ → 用P9反馈
- 某个数字看起来不对？ → 在相关页面截图并说明具体位置
- 功能建议？ → 直接告诉我们

### Q5: 为什么同一条舆论在不同页面显示的内容不一样？
**A**: 通常不会。如果发现差异，这可能是：
- 某页面进行了额外过滤（如P3只显示高风险舆论）
- 数据缓存未更新（刷新页面即可）

### Q6: 我想对某条舆论添加自己的注解/标签，可以吗？
**A**: 在P9互动分析工具中可以。但目前这些自定义标签只在本地保存，不会同步到其他页面。

## 📱 使用技巧

### 技巧1：快速定位问题
```
问题："为什么负面舆论这么多？"

解决方案：
  1. P1看一眼负面舆论的具体话题
  2. P7看这些话题的敏感度
  3. P5看是哪个参与方引发了这些负面舆论
  4. P2搜索相关舆论，找几条代表性的来读
  5. 在P6制定对应的政策措施
  
总耗时：15分钟
```

### 技巧2：定期舆论监测
```
建议每周固定查看：
  周一：P1（看上周数据总结）
  周二：P7（看新出现的敏感话题）
  周三：P3（看风险舆论有无增加）
  周四：P2（搜索重点关键词，看最新舆论）
  周五：P6（整理周报建议）
  
总耗时：30分钟/周
```

### 技巧3：数据验证
```
如果你对某个数据指标有疑问：
  1. 在P2用筛选器精确查询
  2. 看搜索结果的总数（这是最准确的）
```
